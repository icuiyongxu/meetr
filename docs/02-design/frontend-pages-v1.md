# Meetr 前端页面与菜单结构（V1）

## 1. 页面结构
### 用户端
- 登录页
- 首页 / 今日会议
- 会议室查询页
- 日历预定页
- 我的预约
- 预约详情页

### 管理端
- 楼栋管理
- 会议室管理
- 预约规则管理
- 预约管理

## 2. 菜单建议
### 一级菜单
- 工作台
- 会议室预定
- 我的预约
- 管理后台

### 管理后台二级菜单
- 楼栋管理
- 会议室管理
- 预约规则
- 预约记录

## 3. 核心页面说明
### 工作台
- 今日会议
- 即将开始会议
- 快速发起预约入口

### 会议室预定
- 按时间/楼栋/容量筛选
- 卡片或表格展示可用会议室
- 支持进入日历视图预约

### 我的预约
- 我发起的预约
- 状态筛选
- 支持取消/编辑

### 会议室管理
- 会议室列表
- 启停状态
- 容量、设备、位置维护

## 4. 前端组件建议
- 时间范围选择器
- 会议室状态 Tag
- 日历视图组件
- 预约表单弹窗
- 冲突提示弹窗
