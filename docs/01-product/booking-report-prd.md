# 预约报表与使用率统计 — 需求文档

## 1. 背景与目标

会议室系统不仅是“预约工具”，也是“资源管理工具”。管理者和运营人员通常关心：

- 哪些会议室最常用/最闲置
- 某个时间段的使用率如何
- 哪些团队/人预约最多
- 预约通过率、取消率是多少

当前 Meetr 缺少这一层能力，需要补齐一套基础报表，供管理员和运营侧使用。

---

## 2. 目标用户

| 用户 | 诉求 |
|------|------|
| 系统管理员 | 查看全局使用率、识别闲置会议室 |
| 楼栋管理员 | 查看本楼栋各会议室的使用情况 |
| 运营/HR | 了解各部门预约习惯、导出数据 |

---

## 3. 功能范围

### 3.1 会议室使用率报表（核心）

展示每个会议室的使用率统计。

**统计周期**：可按天 / 周 / 月查询

**每个会议室统计维度**：
- 预约次数（有效预约，不含取消）
- 使用总时长（小时）
- 使用率 = 有效预约时长 / 可用总时长
- 取消次数
- 待审批数量（期末时点）

**排序方式**：按使用率降序 / 按使用次数降序

---

### 3.2 预约记录查询（核心）

支持多条件筛选 + 分页展示。

**筛选条件**：
- 楼栋（多选）
- 会议室（多选）
- 预约人（模糊搜索工号/姓名）
- 预约状态（已确认 / 已取消 / 审批中）
- 审批状态（已通过 / 已驳回 / 待审批 / 无需审批）
- 时间范围（开始时间）

**列表字段**：
- 主题
- 楼栋
- 会议室
- 预约人
- 开始时间
- 结束时间
- 时长（分钟）
- 预约状态
- 审批状态
- 创建时间

**操作**：
- 导出 Excel（当前页 or 全部）

---

### 3.3 用户维度统计（建议做）

按用户统计预约行为。

**统计维度**：
- 用户名 / 工号
- 预约次数
- 有效预约次数
- 取消次数
- 预约总时长
- 驳回次数

**排序**：按预约次数降序

---

### 3.4 时间热力分布（建议做）

展示一天中各时段（08:00–22:00）被预约的密集程度。

可按会议室维度或全局聚合。

---

## 4. 页面结构

建议在管理端新增「数据报表」菜单项，包含：

```
数据报表
├── 使用率统计   ← 会议室维度
├── 预约记录     ← 明细查询 + 导出
└── 用户统计     ← 用户维度（建议做）
```

---

## 5. 数据口径说明

| 场景 | 是否计入 |
|------|---------|
| 状态 = BOOKED 且审批通过或无需审批 | ✅ 计入使用率 |
| 状态 = CANCELED | ❌ 不计入使用率，计入取消数 |
| 状态 = PENDING（审批中） | ❌ 不计入使用率 |
| 审批被驳回 | ❌ 不计入 |

---

## 6. 验收标准

- [ ] 会议室使用率报表可按天/周/月切换
- [ ] 每个会议室显示：次数、时长、使用率、取消数
- [ ] 预约记录支持多条件筛选
- [ ] 预约记录支持导出 Excel
- [ ] 用户维度统计可查看（建议做）
- [ ] 时间热力分布可查看（建议做）
