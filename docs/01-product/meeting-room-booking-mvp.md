# 会议室预定系统 MVP 范围

## 目标
基于 MRBS 的业务模型，使用 Spring Boot + Vue3 + Element Plus 重写会议室预定系统，优先交付企业内部可落地的 MVP。

## 一期范围
### 基础主数据
- 园区/地点
- 楼栋/楼层
- 会议室
- 会议室容量、设备、状态

### 预约核心
- 查询空闲会议室
- 创建预约
- 修改预约
- 取消预约
- 预约详情
- 时间冲突校验

### 用户能力
- 我的预约
- 今日会议
- 即将开始会议列表

### 管理能力
- 会议室管理
- 可预约时间规则配置
- 审批开关

## 暂不纳入一期
- 小程序/App
- 门禁/硬件联动
- 复杂工作流引擎
- 多租户复杂化
- 高级报表与 BI
