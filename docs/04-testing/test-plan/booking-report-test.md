# 预约报表与使用率统计 — 测试方案

> 对应功能：会议室使用率报表、预约记录查询 + 导出、用户维度统计

---

## 1. 测试前置条件

1. 数据库已初始化
2. 已有若干预约记录（含取消、审批通过、待审批等）
3. 管理员账号可登录

---

## 2. 功能用例

### 用例 A：会议室使用率报表

**步骤**
1. 管理员进入「数据报表 → 使用率统计」
2. 选择时间范围（今天 / 本周 / 本月）
3. 选择楼栋（可选）

**预期**
- 显示会议室使用率列表
- 每行：会议室名、预约次数、总时长、使用率%、取消数
- 按使用率降序排列

---

### 用例 B：预约记录查询

**步骤**
1. 进入「数据报表 → 预约记录」
2. 筛选条件：
   - 楼栋 = 某楼
   - 状态 = 已取消
   - 时间范围 = 今天
3. 点击搜索

**预期**
- 列表仅显示匹配条件的记录
- 各字段正确显示

---

### 用例 C：导出 Excel

**步骤**
1. 在预约记录页面点击「导出 Excel」
2. 浏览器下载文件

**预期**
- 下载 `.xlsx` 文件
- 文件内容与列表一致

---

### 用例 D：用户维度统计

**步骤**
1. 进入「数据报表 → 用户统计」
2. 选择时间范围

**预期**
- 显示各用户的预约次数、有效次数、取消次数、总时长

---

## 3. 验收清单

- [ ] 使用率报表可按时间范围筛选
- [ ] 使用率计算逻辑正确（不含取消、不含审批中）
- [ ] 预约记录多条件筛选正确
- [ ] 导出 Excel 文件可下载且内容正确
- [ ] 用户统计维度正确
